## 🏁 Getting Started <a name = "getting_started"></a>

### Installing
After clonning the branch, you need to install dependencies.

To install dependencies, run

```
yarn
```

_or_

```
npm install
```

Then, to run locally, you need to run

```
yarn develop
```

_or_

```
npm run develop
```
